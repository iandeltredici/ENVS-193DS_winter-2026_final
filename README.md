# ENVS-193DS_winter-2026_final
Final assignment for ENVS 193DS Winter 2026

## General Information
This is the final assignment for ENVS 193DS Winter 2026 at UCSB. It contains multiple data visualizations, statistical analyses, and interpretations. This is my personal repo and I am the sole contributor. 

## Data and file overview
All code files are in the 'code' folder, all 3 data sets (reading, kelp, MOPL) are located in the 'data' folder. The .qmd file in the 'code' folder contains 4 problems, including 2 visualizations, a generalized linear model and a Kruskal-Wallis test.

The files include code for all models, visualizations, and test in addition to assumption checks, exploratory visualizations, captions for each figure, written interpretations, and responses.
